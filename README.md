# 📊 Retail Customer Behavior Analytics Dashboard

An interactive data analytics dashboard built with **Python**, **Streamlit**, **Pandas**, and **Plotly** to analyze customer shopping behavior and business trends. This project enables users to explore customer purchasing patterns through dynamic filters, interactive charts, and key performance indicators (KPIs).

## 💼 Business Problem

Retail businesses collect large amounts of customer purchase data but often struggle to extract meaningful insights.

This dashboard helps businesses:

- Understand customer purchasing behavior
- Identify high-performing product categories
- Analyze seasonal shopping patterns
- Monitor customer satisfaction
- Support data-driven business decisions

## ▶️ Installation

### Clone the repository

```bash
git clone https://github.com/Gulsan-Yadv/Customer-Behavior-Analytics-Dashboard.git
```

### Navigate to the project

```bash
cd Customer-Behavior-Analytics-Dashboard
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```
